proyek ini adalah kalkulator yang 
Interface Kalkulator:
● Display untuk menampilkan angka dan hasil
● Button untuk angka 0-9
● Button untuk operasi (+, -, ×, ÷)
● Button Clear (C) dan Clear Entry (CE)
● Button equals (=)
● Button decimal point (.)
Functionality:
● Operasi aritmatika dasar (tambah, kurang, kali, bagi)
● Perhitungan berantai (misal: 5 + 3 × 2)
● Penanganan error (pembagian dengan nol)
● Keyboard support (angka dan operator)
Advanced Features:
● History perhitungan (5 perhitungan terakhir)
● Memory functions (M+, M-, MR, MC)
● Responsive design untuk mobile dan desktop
